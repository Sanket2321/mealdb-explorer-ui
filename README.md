## **MealDB Explorer UI**  
**MealDB Explorer UI** is a React + Vite frontend for browsing meals using **TheMealDB API (test key `1`)**.  
It connects to the backend via **REST API calls**, supports **meal search**, **meal details**, **categories**, **areas**, and **random meals**.  
The app runs locally using **`npm install`** and **`npm run dev`** on **http://localhost:5173**, with the backend expected at **http://localhost:8080/api**.
